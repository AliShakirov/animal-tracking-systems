# Animal Tracking System

## 1. Описание проекта
**Animal Tracking System** — это учебный проект, разработанный для демонстрации навыков веб‑разработки. Приложение позволяет отслеживать перемещения животных, хранить данные о местоположении и отображать их через API или визуальный интерфейс (в зависимости от реализации команды). Проект предназначен для студентов, изучающих основы backend и frontend‑разработки.

Основные функции:
- хранение информации о животных;
- запись координат и времени последнего обнаружения;
- API для получения данных;
- возможность расширения до геолокационного интерфейса.

## 2. Установка и запуск
### Требования
- Node.js версии 16+
- npm версии 8+
- ОС: Windows / Linux / macOS

### Установка проекта
```bash
git clone https://github.com/USERNAME/animal-tracking-system.git
cd animal-tracking-system
npm install
```

### Запуск проекта
```bash
npm start
```
После успешного запуска сервер будет доступен по адресу:
```
http://localhost:3000
```

## 3. Примеры использования
### Получение списка животных
Запрос:
```
GET /api/animals
```
Ответ:
```json
[
  {
    "id": 1,
    "name": "Wolf-21",
    "lastLocation": {
      "lat": 55.7522,
      "lon": 37.6156,
      "timestamp": "2025-11-27T18:00:00Z"
    }
  }
]
```

### Добавление местоположения
```
POST /api/animals/1/location
{
  "lat": 55.75,
  "lon": 37.61
}
```
